# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Msharma1212/z-meet.git
cd z-meet
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create:

```env
.env
```

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
GEMINI_API_KEY=your_api_key
```



# ▶️ Run Development Server

```bash
npm run dev
```
